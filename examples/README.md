# Egitsel Ornekler

Bu klasordeki kodlar VALRAVN IDS uretim kodunun kopyasi degildir. Mimariyi aciklamak icin az sayida alanla yazilmis bagimsiz orneklerdir.

- `cascade_decision.py`: RF gri bolgesi ve TabNet yonlendirmesi.
- `window_metrics.py`: zaman penceresinde temel hiz ve cesitlilik olcumu.
- `sample_flows.csv`: tamamiyla sentetik, kimliksiz ornek akislar.
- `flow_schema.json`: ornek alanlarin aciklamasi.

```powershell
python examples\cascade_decision.py
python examples\window_metrics.py
```
